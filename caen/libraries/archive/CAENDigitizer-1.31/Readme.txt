 Content 
 -------

 Readme.txt        			: This file.

 ReleaseNotes.txt  			: Release Notes of the last software release.
 
 lib                        : Directory containing the library binary file
                              and an install script .
 include                    : Directory containing the relevant header files.
 
 samples					: Directory with two c files that show how acquire with this library
 

 System Requirements
 -------------------
 - Linux Kernel 2.4/2.6
 - CAENComm library version 1.01 or above.


 Installation notes
 ------------------
 
 - Run the install script from lib folder:
 
 - Run install for Linux 32 bit platform
 - Run install_64 for Lionux 64 bit platform
